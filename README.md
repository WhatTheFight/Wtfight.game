# WTF.Game

**Your opinion has a price. Prove you’re right.**  
Season-based debate arena on **Solana**, launched in **MVP** on **Pump.fun**.

**Live site (GitHub Pages):**
- `whatthefight.github.io/wtf.game/`

## What is it?
WTF.GAME is an experimental arena where **two camps** face off across a season.
Your side is your stance. The scoreboard is public. The settlement is transparent.

## MVP (Season 1)
- **2 tokens** (one per camp) created on Pump.fun
- **2 Chests** (public multisig wallets)
- Weekly judgment based on:
  - **TWAP (50%)**
  - **Chest weight (50%)**
- Redistribution target: **48 / 48 / 4** (winners / charity / ops)
