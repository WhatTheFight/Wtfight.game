# Contributing

Thanks for your interest in WTFight.

## How to contribute
- Open an Issue for bugs or ideas
- Keep proposals short and practical
- Prefer small PRs that are easy to review

## Ground rules
- No private keys / seed phrases / sensitive data
- No doxxing / harassment
- No promises of financial returns

## Repo layout
- Landing website: `/docs`
- Images for README: `/assets`
- Whitepaper drafts: `/whitepaper`
